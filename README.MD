GreenScene

Find Your Place on Campus.

GreenScene is a web-based campus event application platform designed to support students in exploring and participating in university events. The platform serves as a centralized hub where students, organizations, and administrators can connect through campus activities, workshops, gatherings, club meets, and events that are sponsered by the university.

-----------------------------------------------------------------------------------------------------------------------------------

Project Overview:

Students miss a multitude of opportunities to adapt with the campus life because event information is scattered across social media platforms, flyers, emails, and even organization websites. GreenScene addresses this issue by providing a single platform where users can discover events, see recommendations, and stay informed about upcoming campus activities. This system is being developed as part of the CSCE 3444 Software Engineering course.

-----------------------------------------------------------------------------------------------------------------------------------

Key Features:

Event Discovery
 - Browse upcoming campus events
 - Search and filter events using tags and keywords
 - View detailed event information
 
Event Management
  - Authorized users can create and submit events
  - Administrative approval workflow for event publication
  - Edit and manage event listings
  
RSVP System
  - Register interest in events
  - Save events for future reference
  - Track upcoming activities

AI Chatbot Student Assistant
  - Conversational chatbot for event discovery
  - Personalized event recommendations
  - Assistance for students seeking campus resources

Location Integration
  - Event locations displayed within event details
  - Google Maps integration for navigation assistance
  
Mobile-Friendly Design
  - Responsive interface for desktop and mobile devices
  - Progressive Web App (future enhancement)

User Roles:

Student
  - Browse events
  - Search and filter activities
  - RSVP to events
  - Save favorite events
  - Interact with the AI Assistant

Authorized Event Organizer
  - Create event listings
  - Edit submitted events
  - Manage organization activities

Administrator
  - Approve or reject event submissions
  - Manage users and content
  - Moderate platform activity
